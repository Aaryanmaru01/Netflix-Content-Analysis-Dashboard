# Netflix-Content-Analysis-Dashboard
# 📊 Netflix Data Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing Netflix content data to extract meaningful insights such as content growth, top countries, and popular genres.  
An interactive dashboard is created using Power BI to present the data visually.

---

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution  
- Identify top 5 countries producing content  
- Understand content growth over time  
- Find most popular genres  

---

## 📂 Dataset
- Source: Netflix Titles Dataset  
- Contains:
  - Title
  - Type (Movie/TV Show)
  - Country
  - Date Added
  - Genre
  - Release Year  

---

## 🛠️ Tools Used
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning)

---

## 📊 Dashboard Features
- ✅ Total Shows, Movies & TV Shows KPI
- 📈 Content Added Over Time (Trend Analysis)
- 🌍 Top 5 Countries
- 🍩 Movies vs TV Shows Distribution
- 🎬 Top 5 Genres
- 📅 Date Filter (Slicer)

---

## 🔍 Key Insights
- Most content is produced in the United States
- Movies dominate over TV Shows
- Content growth increased rapidly after 2015
- Drama and Comedy are most popular genres

1. Download the dataset

3. Refresh data if needed
4. Explore using filters and visuals
